10Q_folder:
	- contains all 10Q information downloaded from the SEC financial statements database.
	- https://www.sec.gov/data-research/sec-markets-data/financial-statement-data-sets
	- contains intermediate files used when creating the cleaned data set.
price_batches:
	- contains files which collectively contain historical stock pricing data for stocks since at least 2009.
	- https://finance.yahoo.com/
sic_codes:
	- contains the Standard Industrial Classification codes.
	- https://www.sec.gov/search-filings/standard-industrial-classification-sic-code-list
all_stock_tickers:
	- contains a list of all US tickers and their market caps.
	- https://stockanalysis.com/stocks/
